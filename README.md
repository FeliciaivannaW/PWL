# Pemrograman Web Lanjut (PWL)

Repository ini berisi kumpulan materi, latihan, dan implementasi project selama perkuliahan **Pemrograman Web Lanjut (PWL)** Semester 4.

## 📚 Materi yang Dipelajari

### P01 – P06 : Laravel
Pada pertemuan awal mempelajari framework Laravel, meliputi:
- Routing
- Controller
- Blade Template
- Migration & Seeder
- Database Integration
- CRUD Application
- Authentication dasar
- MVC Concept

### P07 : Node.js
Mempelajari dasar backend menggunakan Node.js:
- Pengenalan Node.js
- Module & NPM
- HTTP Server
- File System
- Routing sederhana

### P10 : Express.js
Mempelajari framework Express.js untuk pengembangan web backend:
- Express Routing
- Middleware
- Static Files
- Form Handling
- Template Engine (EJS)
- Request & Response
- REST API dasar
